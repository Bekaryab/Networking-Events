# Networking & Events 🚀

**Networking & Events** is a modern iOS application designed to aggregate information about IT, business, and student events across Kazakhstan into one centralized platform.  
The app allows users to discover new events, create their own, and connect directly with other participants.

---

## Features

### Authentication & Registration
Secure sign-up and login powered by Firebase Authentication.

### Event Feed & Search
Browse all available events and search them by name.

### Event Creation
Users can create new events with full details including:
- Image
- Title
- Location
- Time
- Guest limit
- Description

### User Profile
A personal profile page where users can manage:
- My Events  
- Liked Events  
- Saved Events  

### Real-time Chat
Search for other users and communicate using Firebase Firestore-based real-time messaging.

### Local Data Storage
User profile images and recent chats are stored locally (UserDefaults & FileManager) for better offline access.

---

## Tech Stack

- **Language:** Swift  
- **UI:** SwiftUI  
- **Architecture:** MVVM  
- **Backend:** Firebase (Authentication, Firestore)  
- **Async:** Combine  
- **Media:** PhotosUI
